== Description ==
Advanced Instagram Carousel is easy way to integrate your Instagram photos/feeds into your WordPress site. 

= Features =

* Advanced Instagram Carousel gives you an option to choose photos with diffrent resolutions.
* Advanced Instagram Carousel is responsive and aditionaly you can add your own HTML Markups.
* Advanced Instagram Carousel supports pre designed themes.
* Advanced Instagram Carousel gives you an option to store feeds into cache to optimize speed.
* You can use your own Instagram API keys. Visit: https://www.instagram.com/developer/
* Advanced Instagram Carousel allows you to customize your photos/feeds with tons of options.

== Installation ==
1. Upload the plugin files to the `/wp-content/plugins/advanced-instagram-carousel` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
